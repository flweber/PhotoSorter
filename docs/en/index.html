Photo Sorter

Photo Sorter is a tool which helps you to sort your holiday pictures in folders so that you can find your photos years ago.

Installation

Use the Installer

Use the Download link for the .msi file. When the download has been finished execute the file you can always click next on every step.
Attention: If you change the installation directory please ensure that the installation directory is not protected.

Manual Installation

Use the Download link for the .zip file. Extract the files in a directory of your choice and run the application.
Attention: The installation directory must be writable for the application.

Usage

First you have to select the daterange of your holidays. In the next step you have to select the source directory where you have stored the images. Last but not least you have to select the folder where your photos should be stored when they are sorted. You don't have to select a folder with your destination. The tool will create a folder named by your destination where you can find your sorted pictures after the process finished.

Features

  • current Features
  • Use the creation date of your pictures to sort them
  • you can select a daterange for the images which will be sorted
  • creates a folder for your destination which includes the sorted files
  • coming soon...
  • possibility to delete the unsorted source files during progress
  • possibility to ignore the daterange
