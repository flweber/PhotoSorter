PhotoSorter

Der Foto Sortierer ist ein kleines Programm, mit dem
Sie mühelos Ihre Urlaubsfotos Datumssortiert ablegen können. Dieses Tool dient dazu, dass Sie sich jederzeit in Ihren Erinerungen zurecht finden!

Installation

Automatische Installation

Laden Sie sich den die .msi-Datei über den Download Link herunter und führen Sie diese aus. Hier können Sie sich nun einfach mit dem Weiter-Knopf durchklicken.
Sollten Sie den Installationspfad ändern achten Sie bitte darauf, dass das Installationsverzeichnis nicht schreibgeschützt ist.

Manuelle Installation

Laden Sie sich den die .zip-Datei über den Download Link herunter und entpacken Sie die Datei in einem Ordner.
Achten Sie hierbei unbedingt darauf, dass sich die Dateien nicht in einem schreibgeschützten Verzeichnis befinden.

Verwendung

Zuerst geben Sie den Datumsbereich ein, in dem sich die Fotos befinden, die Sie sortieren wollen. Als nächstes wählen Sie dann den Ordner aus, in welchem sich die zu sortierenden Bilder befinden, danach machen Sie dasselbe für den Ordner, in den diese herein sortiert werden sollen. In diesem Ordner brauchen Sie keinen weiteren Ordner anlegen, dass übernimmt das Programm für Sie. In dem Ordner wird das Programm für Sie einen Unterordner erstellen, welcher nach Ihrem Urlaubsziel benannt ist. In diesen werden die Fotos nun nach ihrem Aufnahmedatum sortiert.

Features

  • aktuelle Features
  • sortiert Ihre Bilder nach dem Datum sortiert in Ordner
  • Auswahl eines Datumsbereichs
  • eingabe eines Hauptordners, in den die Bilder sortiert kopiert werden
  • geplante Features
  • Möglichkeit die Bilder zu kopieren und dann zu löschen
  • Möglichkeit alle Bilder zu Sortieren
